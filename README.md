### Project Overview
A free, open-source tool that runs entirely in your browser (Client-Side), designed to help job seekers create **ATS-compliant resumes**, visually appealing CVs, and professional Cover Letters.

This project prioritizes privacy; all your data is stored locally on your device and is never shared with any third party servers.

### โจ Key Features
1. **ATS Compatibility**: Smart analysis to match your resume keywords with specific Job Descriptions, maximizing your chances of passing automated filters.
2. **100% Privacy (Offline-First)**: Data is stored in your browser's LocalStorage. No external database or server uploads.
3. **Multiple Layouts**:
   - **ATS Version**: Simplified text structure optimized for parsing bots.
   - **Modern Design**: Attractive visual layout for direct sharing or emailing.
4. **Export & Import**: Save your progress as a JSON file for future edits, or export documents as PDF.
5. **Smart Analysis**: Get an instant ATS Score with actionable improvement tips.
6. **Bilingual Support**: Full support for Arabic (RTL) and English (LTR) interfaces and outputs.

### ๐ How to Use
1. **Run**: Simply open the `index.html` file in any modern web browser.
2. **Fill Data**: Use the sidebar to navigate between sections (Basics, Experience, Education...).
3. **Check ATS Score**:
   - Go to "Settings & ATS" tab.
   - Paste the Job Description of the role you are applying for.
   - Click "Extract Keywords" to see your matching score and missing keywords.
4. **Export**:
   - Click "Download All (PDF)" to get your print-ready files.
   - Click "Save JSON" to backup your working data.
